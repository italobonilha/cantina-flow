# 🍽️ CantinaFlow

Sistema web para gerenciamento de fluxo para cantinas universitárias.

## 📖 Sobre o Projeto

O CantinaFlow surgiu para resolver um problema real observado na cantina de uma universidade.
Os pedidos eram registrados manualmente em papel, causando filas, erros e dificuldade no acompanhamento dos pedidos e o objetivo do projeto é digitalizar todo esse processo através de uma aplicação web simples, rápida e intuitiva.
